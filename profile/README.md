# JMP Pro Software — Advanced Statistics from SAS for Data Discovery

## About JMP Pro Software

JMP Pro Software provides a complete advanced statistical discovery environment from SAS Institute for quality engineers, researchers, and data analysts requiring interactive visualization and predictive modeling. JMP Pro Software combines exploratory data analysis, dynamic linking between graphs and data tables, regression analysis (linear, logistic, nonlinear), ANOVA, mixed models, multivariate methods, and design of experiments. JMP Pro Software includes predictive modeling (decision trees, neural networks, random forests, boosted trees, support vector machines) and model comparison. JMP Pro Software offers advanced time series forecasting (ARIMA, exponential smoothing, neural network models), functional data analysis, and text mining. JMP Pro Software is trusted by quality engineers, research scientists, clinical trial statisticians, and consumer packaged goods analysts worldwide.

## Deep Dive — Advanced Analytics Performance Notes
Graph Builder interactive drag-drop assignment, local data filter brushing, and automatic axis scaling algorithms
Decision tree pruning (minimum split size, leaf complexity, cross-validation folds), and variable importance score calculation
Mixed model REML estimation, covariance structure selection (variance components, compound symmetry, autoregressive), and Kenward-Roger degrees of freedom
Design of Experiments (factorial, fractional factorial, response surface, optimal, space-filling, definitive screening), power analysis, and aliasing assessment
